Lenguaje de Manipulación (DML)

¿Cómo Alterar los Datos?

El lenguaje **DML (Data Manipulation Language)** nos permite gestionar el contenido de las tablas que creamos en el parcial anterior.

INSERT

Agrega nuevas filas a la tabla.

UPDATE

Modifica datos existentes (Requiere cuidado con el WHERE).

DELETE

Elimina registros de forma permanente.

Sintaxis DML 
-- Insertar un alumno
INSERT INTO Alumnos (id, nombre) 
VALUES (1, 'Juan Perez');

-- Cambiar nombre
UPDATE Alumnos SET nombre = 'Juan P.' 
WHERE id = 1;

-- ¡CUIDADO! Sin WHERE borras TODO
DELETE FROM Alumnos WHERE id = 1;

Consultas SQL (DQL)

El poder del SELECT

Extraer información específica es la tarea más común. Aprendemos a usar filtros para generar reportes inteligentes.

Operadores Lógicos

AND • OR • NOT • BETWEEN • LIKE

Ordenamiento

ORDER BY (ASC / DESC)

Ejemplo Visual de Selección
1. Identificar Tabla (FROM)
2. Filtrar Filas (WHERE)
3. Elegir Columnas (SELECT)
-- Buscar productos caros
SELECT nombre, precio 
FROM Productos 
WHERE precio > 500 
ORDER BY precio DESC;

-- Buscar alumnos por apellido
SELECT * FROM Alumnos 
WHERE nombre LIKE 'Gomez%';

Semana de Automatización

Disparadores (Triggers)

Programación Automática

Un **Trigger** es un programa que se ejecuta automáticamente cuando ocurre un evento (INSERT, UPDATE, DELETE). Es ideal para llevar auditorías o actualizar inventarios solos.

A

BEFORE: Antes de que el cambio se guarde.

B

AFTER: Después de que el cambio fue exitoso.

-- Actualizar stock al vender
CREATE TRIGGER tr_DescontarStock
AFTER INSERT ON Ventas
FOR EACH ROW
BEGIN
    UPDATE Productos 
    SET stock = stock - NEW.cantidad
    WHERE id = NEW.id_prod;
END;
